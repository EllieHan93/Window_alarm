## 데이터 구조

### `drink_log.json`
저장 위치: `%APPDATA%\TP_alarm\drink_log.json`

```json
{
  "2026-05-12": { "coffee": 3, "tea": 1 },
  "2026-05-11": { "coffee": 1, "tea": 0 }
}
```

- 키: ISO 날짜 문자열 (`YYYY-MM-DD`)
- 값: `{"coffee": N, "tea": N}` (정수 카운트)
- 날짜별 누적 — 신규 날짜는 `{"coffee": 0, "tea": 0}`으로 초기화 후 증가
