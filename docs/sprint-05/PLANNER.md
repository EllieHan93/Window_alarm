# Sprint 05 — 기획서

**스프린트:** 05 / UI·UX 개선  
**기간:** 2026-05-08  
**목표:** 사용자 피드백을 반영한 UI 테마 변경 및 알람 알림 경험 개선

---

## 스프린트 목표

v1.0 릴리스 이후 UX 피드백을 반영한다.  
어두운 테마를 밝은 테마로 교체하고, 모달 팝업 방식의 알림을 비모달 토스트로 전환해  
알림이 뜨는 동안에도 사용자가 다른 작업을 방해받지 않도록 한다.

---

## 이번 스프린트 범위 (In Scope)

| ID | 기능 | 설명 |
|----|------|------|
| UI-01 | 라이트 테마 전환 | 전체 UI를 화이트/라이트 컬러 팔레트로 교체 |
| UI-02 | 비모달 토스트 알림 | grab_set 없는 3초 자동 닫힘 토스트 + Canvas 애니메이션 |
| UI-03 | 알람 미리보기 버튼 | 메인 창 헤더에 즉시 알람 팝업 확인 버튼 추가 |
| UI-04 | 자동 실행 기본 활성화 | start_with_windows 기본값 True + 시작 시 레지스트리 동기화 |

---

## 범위 외 (Out of Scope)

- 알람 로직 변경
- 새 알람 타입 추가
- 빌드/배포 갱신

---

## 완료 조건 (Definition of Done)

- [o] 메인 창 및 다이얼로그 배경이 흰색으로 표시된다
- [o] 알람 발화 시 모달 없이 화면 우하단에 토스트가 표시된다
- [o] 토스트가 3초 후 자동으로 닫힌다
- [o] 토스트에 파문 + 벨 흔들기 애니메이션이 표시된다
- [o] [🔔 알람 미리보기] 버튼 클릭 시 즉시 토스트가 표시된다
- [o] 최초 설치(또는 config.json 신규 생성) 시 자동 실행이 레지스트리에 등록된다
