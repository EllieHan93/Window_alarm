## Canvas 애니메이션 설계

### 파문 (Ripple)
- 3개 원, 위상 오프셋 `i/3` (0, 1/3, 2/3)
- 반지름: `6 + phase × 26` px
- 색상: `#7c3aed` → `#ede9fe` 선형 보간 (`_lerp_color`)
- 선 두께: `max(1, 2.5 × (1 - phase))`
- 주기: 900ms

### 벨 흔들기
- 텍스트: `🔔`, 폰트 20pt
- y 오프셋: `math.sin(elapsed_ms / 180) × 2` px

### 프로그레스 바
- `bar_fill.place(relwidth = max(0, 1.0 - t))`
- `t = elapsed_ms / 3000`, 선형 감소
