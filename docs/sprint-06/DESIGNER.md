- ※ 클릭 닫힘은 드래그 인터랙션과 충돌 가능성으로 버튼 방식 유지

## TOAST-03: Fade-in 효과

- 등장 시: `-alpha` 속성을 `0.0 → 1.0`으로 250ms 동안 전환
- 프레임 간격: 25ms, 스텝: 10 (총 250ms)
- Fade-in 완료 후 카운트다운 애니메이션 시작

```
t=0ms   alpha=0.0
t=25ms  alpha=0.1
...
t=250ms alpha=1.0  → 4초 카운트다운 시작
```

## EXTRA-05/06: 투명 배경 & 테두리 없음

### 투명 배경
- `_TRANSPARENT_COLOR = "#010101"` (chroma key)
- Canvas `bg = _TRANSPARENT_COLOR`
- `dialog.wm_attributes("-transparentcolor", _TRANSPARENT_COLOR)`
- 효과: 캔버스 빈 영역 → 바탕화면 투과, PNG 투명 픽셀 → 투과

### 테두리 없음
- `outer` 프레임(`bg="#d1d5db"`, padx=1, pady=1) 제거
- `inner` 프레임을 `dialog`에 직접 pack

## PNG 이미지 배치

- 경로: `src/source/Image.png`
- 로드: PIL `Image.open().convert("RGBA").thumbnail()`
- 최대 크기: `(_W-20) × (_CANVAS_H-70)` = 340×126
- 위치: Canvas 하단 중앙 (`anchor="center"`, `img_y = H - height//2 - 4`)
