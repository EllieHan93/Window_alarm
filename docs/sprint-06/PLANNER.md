# Sprint 06 — 기획서

**스프린트:** 06 / 토스트 & 트레이 디자인 개선  
**기간:** 2026-05-08  
**목표:** 알람 토스트 시각적 완성도 향상 및 트레이 아이콘·툴팁 정보 개선

---

## 스프린트 목표

Sprint 05에서 완성된 비모달 토스트와 트레이 기능을 사용자 관점에서 더 세련되게 다듬는다.  
토스트는 알람 타입별 색상 구분과 애니메이션을 강화하고,  
트레이 아이콘은 실제 시침이 움직이는 동적 아이콘으로 교체하며 툴팁 갱신을 빠르게 한다.

---

## 이번 스프린트 범위 (In Scope)

| ID | 기능 | 설명 |
|----|------|------|
| TOAST-01 | 타입별 accent 색상 | 고정=보라(`#7c3aed`), 인터벌=파랑(`#2563eb`) |
| TOAST-02 | 토스트 클릭 닫힘 | 토스트 본문 클릭 시 즉시 닫힘 |
| TOAST-03 | Fade-in 효과 | 등장 시 0→1 알파 전환 (0.3초) |
| TRAY-01 | 동적 시계 아이콘 | 실제 현재 시각 시침/분침으로 아이콘 갱신 |
| TRAY-02 | 툴팁 갱신 주기 단축 | 30초 → 5초 |
| TRAY-03 | 툴팁 정보 보강 | 활성 알람 수, 날짜 포함 |

### 스프린트 중 추가 기획 (범위 확장)

| ID | 기능 | 배경 |
|----|------|------|
| EXTRA-01 | 고양이 Canvas 애니메이션 | 사용자 요청: 애니메이션 캐릭터 형태 알람 |
| EXTRA-02 | PNG 이미지 교체 | 사용자 제공 이미지로 교체 |
| EXTRA-03 | 플로팅 창 + 드래그 | 토스트가 아닌 자유 이동 창 |
| EXTRA-04 | 4초 자동 닫힘 복원 | 카운트다운 바 재추가 |
| EXTRA-05 | 캔버스 투명 배경 | `-transparentcolor` 적용, 바탕화면 투과 |
| EXTRA-06 | 창 테두리 제거 | outer 프레임 제거 |

## 완료 조건 (Definition of Done)

- [o] 고정 알람 토스트 상단 바가 보라색으로 표시된다
- [o] 인터벌 알람 토스트 상단 바가 파란색으로 표시된다
- [ ] 토스트 본문 클릭 시 닫힌다 ← 드래그로 대체, 다음 스프린트 검토
- [o] 토스트 등장 시 페이드인 된다
- [ ] 트레이 아이콘 시침/분침이 현재 시각을 가리킨다 ← Sprint 07 이관
- [ ] 트레이 아이콘이 5초마다 갱신된다 ← Sprint 07 이관
- [ ] 툴팁에 활성 알람 수와 날짜가 표시된다 ← Sprint 07 이관
- [o] PNG 이미지 기반 알람 창이 표시된다
- [o] 창을 드래그로 이동할 수 있다
- [o] 캔버스 배경이 투명하다
