## ON/OFF 토글 버튼 위치

```
[고정 시각 알람 탭]
┌────────────────────────────────────┐
│ [+ 추가]  [수정]  [삭제]  [ON/OFF] │  ← 버튼 행
├────────────────────────────────────┤
│ ON │ 이름       │ 시각  │ 요일 │ 소리 │
│ ✔  │ 업무 시작  │ 09:00 │ 매일 │ ...  │  ← 선택 시 ON/OFF 동작
│ ✘  │ 점심 알림  │ 12:00 │ 평일 │ ...  │
└────────────────────────────────────┘

[인터벌 알람 탭]
┌────────────────────────────────────┐
│ [+ 추가]  [수정]  [삭제]  [ON/OFF] │
├────────────────────────────────────┤
│ ON │ 이름     │ 간격  │ 소리 │
│ ✔  │ 휴식 알림 │ 1시간 │ beep │
└────────────────────────────────────┘
```

- 선택 없이 ON/OFF 클릭 → 무시 (선택 행 없으면 동작 안 함)
- 토글 즉시 저장 + 목록 갱신 + 선택 행 유지
- 스타일: 기본 `TButton` (파란 배경) — 별도 색상 없음
