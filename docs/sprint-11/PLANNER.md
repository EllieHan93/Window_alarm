# Sprint 11 — PLANNER

**스프린트:** 11 / UI 이모티콘 제거 & 알람 개별 ON/OFF 토글  
**목표:** 렌더링 불가한 이모티콘을 UI 전체에서 제거하고, 관리자 화면의 각 알람에 개별 활성화 토글 버튼을 추가한다.

---

## 배경

Sprint 10에서 Mona12 픽셀 폰트로 교체한 이후 이모티콘이 렌더링되지 않는 문제가 확인됨.  
GDI 방식으로 등록된 ColorEmoji.ttf가 tkinter(DirectWrite)에서 자동 폴백되지 않기 때문.  
폰트를 유지하면서 이모티콘을 한국어 텍스트로 교체하는 방식으로 해결.

또한 각 알람의 활성화 여부를 목록 화면에서 즉시 토글할 수 있는 버튼이 필요함.  
기존에는 수정 다이얼로그를 열어 체크박스를 변경해야 했음.
