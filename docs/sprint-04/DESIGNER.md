# Sprint 04 — 디자인 사양서

**스프린트:** 04 / 빌드 및 배포  
**기간:** 2026-05-08

## 설치 경험 설계

### 설치 마법사 흐름

```
[1. 환영 화면]
  "TP Alarm 설치를 시작합니다"
  → [다음]

[2. 설치 옵션]
  ☐ 바탕화면 바로가기 만들기
  ☑ Windows 시작 시 자동 실행  (기본 체크)
  → [다음]

[3. 설치 중]
  진행 바

[4. 완료]
  ☑ TP Alarm 시작하기
  → [마침]
```

### 설치 후 시작 메뉴 구성

```
시작 메뉴 > TP Alarm
  ├── TP Alarm (앱 실행)
  └── TP Alarm 제거
```

---

## 앱 아이콘 (.ico) 사양

| 속성 | 값 |
|------|-----|
| 포맷 | ICO (다중 해상도) |
| 포함 크기 | 16, 32, 48, 64, 128, 256 px |
| 디자인 | 보라색 원형 배경 + 흰색 시계 |
| 생성 방법 | Pillow 스크립트 (`make_icon()`) |
| 저장 위치 | `assets/icon.ico` |

---

## 배포 파일 사양

| 파일 | 크기 | 설명 |
|------|------|------|
| `TP_Alarm_Setup.exe` | ~22 MB | 최종 배포 파일 (런타임 포함) |
| `dist\TP_Alarm\TP_Alarm.exe` | ~5.6 MB | 앱 본체 |
| `dist\TP_Alarm\_internal\` | — | Python 런타임 라이브러리 |

---

## 설치 경로

```
%LocalAppData%\TP Alarm\
```

사용자 개인 폴더에 설치 → 관리자 권한 불필요.

---

## 설치 시 선택 가능 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| 바탕화면 바로가기 | 미체크 | 선택적 생성 |
| Windows 시작 시 자동 실행 | 체크 | `%APPDATA%\Roaming\Microsoft\Windows\Start Menu\Programs\Startup` 바로가기 |
