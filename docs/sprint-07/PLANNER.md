# Sprint 07 — 기획서

**스프린트:** 07 / 관리자 잠금 & 트레이 개선  
**기간:** 2026-05-12  
**목표:** 알람 설정을 PIN으로 보호하고, 트레이에서 열면 최소 뷰(사용 시간)만 표시

---

## 스프린트 목표

알람 관리 UI를 관리자 PIN(4자리)으로 잠가 일반 사용 중에는 노출되지 않도록 한다.  
트레이에서 창을 열면 사용 시간만 보이는 미니 상태 뷰가 뜨고,  
[관리자 설정] 버튼 → PIN 입력 → 관리 뷰 진입 구조로 변경한다.  
Sprint 06에서 이관된 트레이 동적 아이콘·툴팁도 함께 완성한다.

---

## 이번 스프린트 범위

| ID | 기능 | 설명 |
|----|------|------|
| AUTH-01 | PIN 저장 | `AppSettings.admin_pin` 필드, 기본값 `"0104"` |
| AUTH-02 | PIN 입력 다이얼로그 | 4자리 입력, 3회 실패 시 자동 닫힘 |
| UI-01 | 상태 뷰 (기본) | 사용 시간 + 다음 알람 + [관리자 설정] 버튼만 표시 (320×200) |
| UI-02 | 관리 뷰 (PIN 인증 후) | 기존 탭 UI + [🔒 잠금] 버튼 (520×560) |
| TRAY-01 | 동적 시계 아이콘 | 현재 시각 시침/분침 반영 |
| TRAY-02 | 툴팁 갱신 5초 | 30초 → 5초 |
| TRAY-03 | 툴팁 정보 보강 | 날짜 + 활성 알람 수 포함 |

---

## 범위 외

- PIN 변경 UI (다음 스프린트)
- 로그인 세션 유지 시간 설정

## 완료 조건 (Definition of Done)

- [o] 트레이에서 창을 열면 사용 시간과 [관리자 설정] 버튼만 표시된다
- [o] [관리자 설정] 클릭 시 PIN 입력창이 뜬다
- [o] 올바른 PIN(0104) 입력 시 관리 뷰로 전환된다
- [o] 관리 뷰에서 [🔒 잠금] 클릭 시 상태 뷰로 돌아간다
- [o] 잘못된 PIN 3회 입력 시 다이얼로그가 닫힌다
- [o] 트레이 아이콘이 현재 시각 시침/분침을 표시한다
- [o] 툴팁이 5초마다 갱신된다
- [o] 툴팁에 날짜와 활성 알람 수가 표시된다
