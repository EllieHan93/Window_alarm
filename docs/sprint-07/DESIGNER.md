```
┌──────────────────────┐
│  🔐  관리자 비밀번호  │
│                      │
│    [ ● ● ● ● ]       │  ← Entry, show="●", 중앙 정렬
│                      │
│  비밀번호가 틀렸습니다 (2회 남음)  ← 빨간 오류 텍스트
│                      │
│   [확인]  [취소]     │
└──────────────────────┘
```

- 창 크기: 자동 (resizable=False)
- 부모 창 중앙에 배치
- Enter 키 = 확인
- 3회 실패: 오류 메시지 1.5초 표시 후 자동 닫힘
