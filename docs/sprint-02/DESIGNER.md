# Sprint 02 — 디자인 사양서

**스프린트:** 02 / 알람 로직  
**기간:** 2026-05-08

---

## 디자인 범위

Sprint 02는 UI가 없는 비즈니스 로직 레이어다.  
이 스프린트의 디자인 결정은 **사용자에게 어떻게 보이는지**에 집중한다.

---

## 사용 시간 표시 형식

| 경과 시간 | 표시 형식 | 예시 |
|-----------|-----------|------|
| 59초 이하 | `M분 SS초` | `0분 45초` |
| 60초 ~ 59분 59초 | `M분 SS초` | `32분 07초` |
| 1시간 이상 | `N시간 MM분` | `2시간 05분` |

→ 초가 표시되는 기준: 1시간 미만일 때. 1시간 이상이면 분 단위까지만 표시.

---

## 다음 알람 표시 형식

```
[알람 이름] (N시간 MM분)
예: 휴식 알림 (29분 45초)
예: 오전 업무 시작 (1시간 03분)
알람 없음
```

---

## 알람 발화 타이밍 사용자 경험

| 상황 | 사용자가 보는 것 |
|------|----------------|
| 고정 알람 1분 전 | 다음 알람 카운트다운이 감소 중 |
| 발화 시각 도달 | 팝업 즉시 표시 (Sprint 03 구현) |
| 인터벌 알람 재설정 | 다음 알람 카운트다운이 `interval_minutes`로 리셋 |

---

## 요일 표시 규칙

| days 값 | 화면 표시 |
|---------|-----------|
| `[]` | 매일 |
| `[0,1,2,3,4]` | 월화수목금 |
| `[5,6]` | 토일 |
| `[1,3]` | 화목 |
