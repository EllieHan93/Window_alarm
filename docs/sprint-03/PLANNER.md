# Sprint 03 — 기획서

**스프린트:** 03 / UI 및 스레딩 통합  
**기간:** 2026-05-08  
**목표:** 사용자가 실제로 사용할 수 있는 완전한 앱 완성

---

## 스프린트 목표

Sprint 01/02의 로직을 시각적 인터페이스와 연결하고,  
3개의 스레드가 안전하게 협력하는 완전한 앱을 완성한다.

---

| ID | 기능 | 설명 |
|----|------|------|
| F-05 | 시스템 트레이 상주 | 트레이 아이콘, 우클릭 메뉴, 툴팁 |

---

## 완료 조건 (Definition of Done)

- [o] 앱 시작 시 트레이 아이콘이 표시된다
- [o] 트레이 클릭 → 메인 창 열림
- [o] 창 X 버튼 → 트레이 상주 (종료 아님)
- [o] 메인 창에서 사용 시간이 1초마다 증가한다
- [o] 알람 팝업이 최상단에 표시되고 소리가 재생된다
- [o] [5분 후 다시] 클릭 시 5분 뒤 팝업 재표시
- [o] 트레이 우클릭 > 종료 시 앱이 정상 종료된다
- [o] Sprint 03 TC 6개 전부 통과
