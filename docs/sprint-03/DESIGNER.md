## 알람 토스트 와이어프레임 (340×148, 화면 우하단)

> **v1.1 변경:** 모달 팝업(grab_set + wait_window) → 비모달 토스트 (auto-close 3초 + 애니메이션)

```
┌──────────────────────────────────────┐  ← 1px 회색 외곽 테두리
│ [████████ 보라 상단 바 4px ██████████] │
│                                      │
│  [🔔 캔버스]  고정 시각 알람  [FG2] │  ← 왼쪽: 파문+벨 애니메이션 60×60
│  [파문 애니]  오전 업무 시작  [bold] │
│             [5분 후 다시] [닫기]     │
│                                      │
│ [████ 보라 프로그레스 바 4px ▓▓▓░░] │  ← 3초 카운트다운 (우→좌 축소)
└──────────────────────────────────────┘
  위치: 화면 우하단 (sw-20, sh-60)
  속성: overrideredirect(True), -topmost
  동작: 3초 후 자동 닫힘, [5분 후 다시] 클릭 시 즉시 닫고 스누즈 예약
```

**애니메이션 상세:**
- 벨(🔔): `math.sin(elapsed_ms / 180)` 픽셀 진동 (좌우 흔들기)
- 파문(Ripple): 3개 원, 각각 1/3 위상 차이, accent→연보라 색 보간, 900ms 주기로 반복
- 프로그레스 바: `relwidth = 1.0 - t` (t: 0→1, 3초 선형)

- 크기: 64×64 RGBA (Pillow 런타임 생성)

- 툴팁: `"TP Alarm\n사용 시간: 2시간 30분\n다음 알람: 휴식 알림 (29분 45초)"`

## 인터랙션 정의

| 인터랙션 | 트리거 | 동작 |
|----------|--------|------|
| 창 열기 | 트레이 좌클릭 또는 "창 열기" | deiconify + lift + focus |
| 창 닫기 | X 버튼 | withdraw (앱 유지) |
| 알람 토스트 | 발화 이벤트 | 우하단 비모달 토스트 표시 (3초 자동 닫힘) |
| 스누즈 | [5분 후 다시] | 즉시 닫힘 + root.after(300000, show) 예약 |
| 앱 종료 | 트레이 우클릭 > 종료 | root.quit() |
| 알람 미리보기 | 헤더 [🔔 알람 미리보기] 버튼 | AlarmNotification.show() 즉시 호출 |
