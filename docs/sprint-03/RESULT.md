## 후속 개선 사항 (Post v1.0 — UI 개선)

**배경:** v1.0 릴리스 후 UX 피드백을 반영해 UI를 개선했다.

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| UI 테마 | 다크 테마 (`#1e1e2e` 배경) | 라이트 테마 (`#ffffff` 배경) |
| 알람 알림 | 모달 팝업 (grab_set + wait_window 블로킹) | 비모달 토스트 (3초 자동 닫힘) |
| 알람 알림 UX | 클릭하기 전까지 다른 작업 불가 | 알림 뜨는 동안 다른 창 조작 가능 |
| 알림 시각화 | 텍스트만 | 파문+벨 Canvas 애니메이션 + 카운트다운 바 |
| 알람 미리보기 | 없음 | 헤더 [🔔 알람 미리보기] 버튼 추가 |
| 자동 실행 기본값 | 비활성화 (수동 설정 필요) | 활성화 (최초 설치 즉시 등록) |

**수정 파일:**
- `src/main_window.py` — 라이트 테마, 미리보기 버튼
- `src/notification.py` — 비모달 토스트, Canvas 애니메이션
- `src/config_manager.py` — `start_with_windows` 기본값 `True`, 시작 시 레지스트리 동기화
