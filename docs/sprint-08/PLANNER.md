# Sprint 08 — 기획서

**스프린트:** 08 / 폰트·UI 정제 & 음료 요청 알림  
**기간:** 2026-05-12  
**목표:** 커스텀 폰트 적용, 상태 뷰 개선, 숨겨진 관리자 버튼, Telegram 음료 요청 알림

---

## 스프린트 목표

교보 손글씨 폰트를 전체 적용해 개성 있는 UI를 완성하고,  
상태 뷰를 넓혀 텍스트 잘림을 해소한다.  
관리자 진입 버튼은 헤더 오른쪽 상단 투명 영역으로 숨기고,  
상태 뷰에 "☕ 커피 마시고싶다" / "🍵 차 마시고싶다" 버튼을 추가해  
누르면 Telegram 봇으로 관리자에게 알림을 전송한다.

---

## 이번 스프린트 범위

| ID | 기능 | 설명 |
|----|------|------|
| FONT-01 | 커스텀 폰트 등록 | `fonts/KyoboHandwriting2025lyb.ttf` → Windows 등록, `"Kyobo Handwriting 2025"` |
| FONT-02 | 전체 폰트 교체 | main_window, notification 모두 커스텀 폰트 적용 |
| UI-01 | 상태 뷰 확대 | 320×200 → 400×240 (텍스트 잘림 해소) |
| UI-02 | 숨겨진 관리자 버튼 | 헤더 오른쪽 투명 클릭 영역 (cursor=arrow, 배경=accent) |
| UI-03 | 커피/차 버튼 | 상태 뷰에 두 버튼 추가 |
| NOTIF-01 | Telegram 알림 전송 | `notifier.py`, urllib 기반, 비동기 daemon 스레드 |
| NOTIF-02 | Telegram 설정 UI | 설정 탭에 봇 토큰 / 채팅 ID 입력 필드 추가 |

---

## 완료 조건 (Definition of Done)

- [o] 앱 전체 폰트가 교보 손글씨로 표시된다
- [o] 상태 뷰 텍스트가 잘리지 않는다 (400×240)
- [o] 헤더 오른쪽 상단 클릭 시 PIN 다이얼로그가 뜬다 (버튼 시각적으로 숨김)
- [o] "☕ 커피 마시고싶다" 버튼 클릭 시 Telegram 메시지가 전송된다
- [o] "🍵 차 마시고싶다" 버튼 클릭 시 Telegram 메시지가 전송된다
- [o] 토큰/채팅ID 미설정 시 앱이 오류 없이 동작한다
- [o] 설정 탭에서 Telegram 봇 토큰 / 채팅 ID를 저장할 수 있다
