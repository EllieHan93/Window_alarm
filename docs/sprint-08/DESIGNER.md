## 상태 뷰 (400×240)

```
┌────────────────────────────────────────┐  400×240
│ ⏰  TP Alarm                      [ ] │  ← 투명 관리자 버튼 (우측 상단)
├────────────────────────────────────────┤
│  사용 시간          다음 알람까지      │
│  1시간 23분         24분 15초          │  ← 교보 손글씨 폰트
├────────────────────────────────────────┤
│   [☕ 커피 마시고싶다] [🍵 차 마시고싶다] │
└────────────────────────────────────────┘
```

### 버튼 색상

| 버튼 | 배경 | 활성 배경 |
|------|------|----------|
| ☕ 커피 | `#7c3aed` (보라) | `#6d28d9` |
| 🍵 차   | `#059669` (초록) | `#047857` |

### 숨겨진 관리자 버튼

- 위치: 헤더 오른쪽
- 배경/전경: `_ACCENT` (= 헤더 배경과 동일)
- 커서: `arrow` (클릭 영역 힌트 없음)
- 텍스트: `""` (빈 문자열)
- 크기: `width=3`

## Telegram 설정 UI (관리 뷰 > 설정 탭)

```
── Telegram 알림 설정 ───────────────────
봇 토큰 (@BotFather에서 발급)
┌──────────────────────────────────────┐
└──────────────────────────────────────┘
채팅 ID (Chat ID)
┌────────────────┐

── Telegram 봇 설정 방법 ─────────────

3. 발급된 토큰을 위 입력란에 붙여넣기
4. 봇에게 메시지 보낸 후 Chat ID 확인
```
