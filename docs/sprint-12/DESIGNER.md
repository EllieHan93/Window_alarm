## 알림 팝업 말풍선 변경

### Before
```
┌─────────────────────┐
│   고정 시각 알람      │  ← 제거
│   알람 레이블 텍스트   │
└─────────────────────┘
```

### After
```
┌─────────────────────┐
│                     │
│   알람 레이블 텍스트   │  ← 수직 중앙 정렬
│                     │
└─────────────────────┘
```

레이블 텍스트 위치: `by1 + bh // 2` (말풍선 높이 중앙)

---

## 알림 팝업 하단 패널 변경

### Before
```
2026/05/13 14:30   |  1시간 23분 사용
[5분 후 다시]  [닫기]   ← 제거
```

### After
```
2026/05/13 14:30   |  1시간 23분 사용
```

버튼 제거 후 `close()` 함수는 4초 자동 닫힘에만 사용됨.

## 음료 피드백 변경

| 항목 | Before | After |
|------|--------|-------|
| 텍스트 색상 | `_ACCENT` / `_GREEN` (배경과 동일) | `_RED` (#E52521) |
| 표시 시간 | 700ms | 1500ms |
