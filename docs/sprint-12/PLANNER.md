# Sprint 12 — PLANNER

**스프린트:** 12 / 알림 팝업 UI 정리 & 음료 피드백 개선  
**목표:** 알람 팝업에서 불필요한 텍스트와 버튼을 제거하고, 음료 요청 시각적 피드백을 개선한다.

---

## 배경

Sprint 11 완료 후 실사용 중 발견된 UX 문제:
- 알림 팝업 말풍선에 "고정 시각 알람" / "인터벌 알람" 타입 텍스트가 불필요하게 표시됨
- "5분 후 다시", "닫기" 버튼이 팝업 하단을 차지해 레이아웃이 복잡함
- 커피/차 버튼 클릭 시 피드백이 너무 미약해 반응이 없는 것처럼 느껴짐

---

## 사용자 스토리

| ID | 스토리 |
|----|--------|
| US-12-1 | 알람 팝업에 알람 타입 텍스트("고정 시각 알람" 등)가 표시되지 않는다. |
| US-12-2 | 알람 팝업에 "5분 후 다시", "닫기" 버튼이 없다. |
| US-12-3 | 커피/차 버튼 클릭 시 빨간 텍스트가 1.5초간 명확하게 표시된다. |
